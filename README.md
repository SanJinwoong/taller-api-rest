# Taller API REST con Node.js y Express

Este proyecto valida los conocimientos adquiridos en el taller de API REST.

**Autor:** Mendez Padron Gustavo Emanuel

## Instrucciones para ejecutar

1.  Instalar dependencias:
    ```bash
    npm install
    ```
2.  Ejecutar el servidor:
    ```bash
    npm run dev
    ```
3.  Abrir en el navegador: `http://localhost:3000`

## Endpoints
* GET `/user`: Ver usuarios.
* POST `/user`: Crear usuario.
* PUT `/user/:id`: Editar usuario.
* DELETE `/user/:id`: Borrar usuario.
